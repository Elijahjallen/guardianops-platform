# GuardianOps MVP Features

## MVP Goal

The first version of GuardianOps will provide a secure case-management workflow for intake, quoting, assignment, expense tracking, and field updates.

---

## Included in MVP

### 1. Authentication
- Employee login
- Field staff login
- Role-based access
- Password hashing

### 2. Parent Intake Form
- Parent/guardian information
- Child/client information
- Pickup location
- Destination facility
- Medical/behavioral notes
- Emergency contact information

### 3. Employee Dashboard
- View all cases
- Filter cases by status
- View active and completed cases
- Open case detail pages

### 4. Case Management
- Create case from intake
- Update case status
- Assign staff
- Add internal notes
- Track case lifecycle

### 5. Quote Builder
- Add airfare cost
- Add hotel cost
- Add ground transportation
- Add meals/per diem
- Add miscellaneous expenses
- Calculate estimated total

### 6. Expense Tracking
- Add expenses to a case
- Categorize expenses
- Track estimated vs actual cost
- Calculate estimated profit

### 7. Field Staff Updates
- Mobile-friendly update screen
- Submit transport status updates
- Add notes during transport
- Timestamp each update

### 8. Notifications
- Notify internal team when case status changes
- Notify parent/guardian when field update is submitted

### 9. Basic Reporting
- Active cases
- Completed cases
- Daily/weekly/monthly case count
- Revenue estimate
- Expense estimate
- Profit estimate

---

## Not Included in MVP

These will be added later:

- Travel booking API integration
- Payment processing
- E-signatures
- Advanced analytics
- Mobile app store version
- AI route/planning assistant
- Automated flight pricing
- Payroll integration

---

## MVP Success Criteria

The MVP is complete when a user can:

1. Submit or enter an intake form
2. Create a case
3. Generate a quote
4. Assign field staff
5. Track case progress
6. Submit field updates
7. Track expenses
8. View dashboard metrics
